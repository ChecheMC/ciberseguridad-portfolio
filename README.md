# Portfolio de Ciberseguridad — Michelzon Montañez

Estudiante de Ingeniería en Conectividad y Redes con orientación a operaciones de seguridad (SOC), detección de amenazas y respuesta a incidentes. Este repositorio reúne documentación de mi aprendizaje y práctica en ciberseguridad defensiva.

> **Nota:** este portafolio documenta conceptos, procesos y aprendizajes con fines profesionales. No incluye código de producción, configuraciones, ni detalles de infraestructura de ningún sistema real.

## Áreas de práctica

### Operaciones de seguridad (SOC)
He desarrollado experiencia práctica montando y operando un entorno de monitoreo de seguridad basado en herramientas open source: gestión de eventos, reglas de detección, análisis de logs y respuesta a incidentes.

### Detección basada en comportamiento
Trabajo con telemetría de procesos mapeada al framework **MITRE ATT&CK** para detectar comportamiento sospechoso (ejecución anómala, persistencia, etc.).

### Monitoreo de integridad de archivos (FIM)
Implementación de verificación de integridad mediante hashes para detectar modificaciones no autorizadas, con enriquecimiento de inteligencia de amenazas.

### Hardening y respuesta a incidentes
Aseguramiento de servidores (firewall, autenticación multifactor, buenas prácticas) y respuesta documentada a vulnerabilidades reales (ver caso de estudio).

### Aprendizaje en seguridad OT/ICS
Estudio activo de seguridad de sistemas de control industrial: PLCs, SCADA, protocolos industriales y casos reales (ver bitácora).
